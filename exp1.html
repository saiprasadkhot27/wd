HTML Text Formatting Tags Example

  • HTML is a markup language used to create web pages.
  • HTML stands for HyperText Markup Language.
  • HTML was created by Tim Berners-Lee.
  • HTML is easy to learn for beginners.
  • Old version: HTML 4 XHTML
    New version: HTML5
  • HTML uses tags to structure content.
    <br> tag is used to break a line.
  • HR tag is used to draw a horizontal line. 
  • HTML Structure:
<html>
  <head>
  </head>
  <body>
  </body>
</html>
